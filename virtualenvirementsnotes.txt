Creating Virtual Environment:

--> python -m venv <env_ name>

Activating virtual environment : 
--> <env_name>\scripts\activate

Deactivating virtual environment
--> <env_name>\scripts\deactivate

Installing Libraries Using pip(pypi)
--> pip install <package_name>
-->pip install <package_name>==version[for specified version] 

Uninstalling Libraries Using pip(pypi)
-->pip uninstall <package_name>
Installing Multiple Libraries Using pip(pypi):
-->pip install -r requirements.txt
